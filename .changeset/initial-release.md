Initial public release of `@youbamj/tree`.

### Highlights

- Full TypeScript rewrite with strict typings.
- Bun-first development workflow.
- Modern CLI (`ytree`) powered by Commander.
- File tree rendering with ignore patterns, depth limits, sorting, hidden-file controls, and optional JSON output.
- Library API for rendering custom object trees.
- Test coverage with Bun test.
- Biome formatting/linting + Husky pre-commit hooks.
- GitHub Actions CI and automated Changesets-based release flow.
